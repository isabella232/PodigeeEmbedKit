    Request embed data information for a podcast. This contains information about the podcast and data for the most recent published episode.

    Declaration

    Swift

    public static func embedDataForPodcastWith(domain: String, episodePath: String? = nil, complete: @escaping (_ result: Result<PodcastEmbed, Error>) -> Void)

    Parameters

    domain

    The domain of the podcast, e.g. bananaland.podigee.io.
    episodePath

    The url path component for the episode, e.g. 63-bbc-sound-sammlung. If set to nil the most recent episode of the podcast will be included in the response.

    The closure called when the network request is finished.

  • playlistForPodcastWith(domain:pageSize:offset:sortBy:complete:)

    Request the episode playlist for a podcast. This returns an array of episodes.

    Declaration

    Swift

    public static func playlistForPodcastWith(domain: String, pageSize: Int = 10, offset: Int = 0, sortBy: PlaylistSorting = .publishDate, complete: @escaping (_ result: Result<Playlist, Error>) -> Void)

    Parameters

    domain

    The domain of the podcast, e.g. bananaland.podigee.io.
    pageSize

    Maximum number of episodes this request should return. Defaults to 10.
    offset

    Offset for paging requests. Defaults to 0.
    sortBy

    Set the sorting of the returned episode playlist. Defaults to sorting by publish date.
    complete

    The closure called when the network request is finished. If successfull you receive an array of episodes.

    Return Value

    Void
