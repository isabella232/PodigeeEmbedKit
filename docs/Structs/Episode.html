  • coverartUrlFor(width:)

    The Podigee image service allows you to request a coverart with a specific size. All coverarts are squares therefore you can only define a the width for the image you need.

    Declaration

    Swift

    public func coverartUrlFor(width: Int) -> URL?

    Parameters

    width

    Define the width of the square coverart you want to request. The height of the image is going to match the width.

    Return Value

    The URL to the coverart image with the requested width.
